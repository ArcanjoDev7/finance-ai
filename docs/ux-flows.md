# Finance AI UX Flows

## First access

1. Create an account or sign in.
2. Land on the dashboard with direct actions for first expense, income, investment or AI command.

## Financial entry

Use the dashboard quick actions or Movimentações. The modal asks for amount and description first, then category. A success toast confirms completion.

## Investments and crypto

Investments are added from the portfolio view. Crypto supports purchase, sale and conversion. The assistant accepts the same natural-language commands and shows the resulting action in conversation.

## Privacy

The topbar visibility control hides financial values across the current app session.
