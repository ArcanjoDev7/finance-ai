# Finance AI

Base arquitetural para o assistente financeiro multiplataforma. O projeto inicia pela web e evolui para Android e iOS sem bifurcar regras de domínio.

Consulte [docs/architecture.md](docs/architecture.md) para a árvore, limites de camadas, fluxo de dependências e decisões técnicas.

Arquitetura, modelagem SQL, infraestrutura Supabase e base Flutter estão versionadas. A conexão a projetos Supabase remotos requer credenciais e OAuth por ambiente; a feature de Login ainda não foi iniciada.
